🤖 Auto-Track Smart Filter

When Auto-Track is enabled, WiFi Mapper works silently in the background. To prevent database spam (e.g., from frequent brief disconnects), it uses a smart cooldown. If you reconnect to the exact same network within 1 hour (configurable in Settings), the connection is ignored. This keeps your timeline clean and saves battery.
