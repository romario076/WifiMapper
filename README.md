Welcome to the **WiFiMapper** repository! This is my very first Android application, built from the ground up to help users easily find and map Wi-Fi networks. Whether you are a traveler looking for a quick connection or just exploring your local area, WiFiMapper is designed to make internet access easier for everyone.

---

## ✨ Features

* **🌍 Interactive Mapping:** Discover Wi-Fi networks in your vicinity.
* **🔐 Secure Authentication:** Seamless and safe login using Google Sign-In.
* **☁️ Cloud Sync:** Powered by Firebase to keep data updated in real-time.
* **🎨 Modern UI:** A beautiful, responsive interface built with Material Design 3 components.
* **🚀 Fast & Lightweight:** Optimized for smooth performance on a wide range of Android devices.

This project leverages modern Android development tools and libraries:

* **UI/UX:** Material Components for Android (M3)
* **Backend:** Firebase (Authentication, Realtime Database/Firestore)
* **Services:** Google Play Services & Google Sign-In
